Application 6: Real-Time System Prototype

This project is a Wokwi-based ESP32 FreeRTOS prototype for a Lockheed Martin style pilot training simulator safety monitor. The system uses real-time scheduling to read a simulated sensor, update safety LEDs, handle an emergency stop interrupt, and send diagnostic output through UART.

The main goal is to prove that hard real-time tasks, such as sensor monitoring and emergency stop handling, complete before their deadlines. The project uses FreeRTOS tasks, an ISR, queues, semaphores, mutexes, and logic analyzer VCD output to demonstrate predictable timing.
